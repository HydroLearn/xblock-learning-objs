Examples

Given a topographic dataset, the student will be able to implement the key steps in watershed delineation, including pitfilling, flow direction, flow accumulation, threshold assignment, and stream network extraction. [Applying]

Given a water supply and demand dataset, the student will be able to assess potential value of using non-traditional water resources to reduce water stresses, considering cost factors associated with the non-traditional resources [Analyzing]

Condition

The condition under which the student will perform the described behavior. Examples include:

  • Given a topographic dataset
  • Given a water supply and demand dataset,
  • Given a diagram of the water cycle,
  • Using historical rain gauge data from NOAA’s Precipitation Frequency Data Server (PFDS),
  • Using the alternating block method and Intensity-Duration-Frequency (IDF) curves,
  • Given instructions on how to use a script language,

Action verb + Object

A learning objective contains a verb that refers to actions associated with the intended learning level from Bloom’s Taxonomy, and an object that describes the knowledge students are expected to acquire or construct. Examples include:

  • Implement the key steps in watershed delineation
  • Assess potential value of using non-traditional water resources to reduce water stresses
  • Interpret at least three sources of ground water recharge
  • Develop Depth-Duration-Frequency (DDF) curves
  • Develop a design-storm hyetograph
  • Construct a simple NWM-based flood warning tool

Bloom's Taxonomy Learning Levels (with examples of action verbs)

  • Remembering: define, describe, identify, label, list, match, name, order, recall, select
  • Understanding: illustrate, compare, distinguish, estimate, explain, classify, interpret, predict, summarize
  • Applying: organize, solve, demonstrate, discover, modify, operate, predict, prepare, produce, relate, solve, implement
  • Analyzing: analyze, break down, compare, select, contrast, deconstruct, discriminate, distinguishes, identify, outline
  • Evaluating: rank, assess, monitor, check, test, judge, Appraise, Criticize, Defend, Conclude, Critique, Justify, Support
  • Creating: generate, plan, compose, develop, create, invent, organize, construct, produce, compile, design, devise

Degree

Describe the degree or level of performance. Examples include:

  • including pitfilling, flow direction, flow accumulation, threshold assignment, and stream network extraction
  • considering cost factors associated with the non-traditional resources
  • with a definition of each source and description of how it functions in the larger water cycle
  • for at least 5 return periods between 2-year to 100-year
  • for a prescribed temporal interval
  • with the ability to notify users when the NWM predicts above average streamflows in a particular region
